مسار الأمن السيبراني بشكل واضح

الأمن السيبراني مو “أداة” وحدها — هو مسار متدرّج. إذا تمشي خطوة خطوة، راح تبني أساس قوي وتفهم الهجمات والدفاع بشكل علمي وعملي.

الطريق المختصر (Roadmap)

  • 1) أساسيات الشبكات: IP / Subnet / DNS / HTTP
  • 2) Linux + Terminal: أوامر، صلاحيات، خدمات
  • 3) برمجة: Python (أتمتة) + أساسيات Web
  • 4) أمن الويب: OWASP Top 10 + جلسات + صلاحيات
  • 5) مختبر عملي: VM + بيئات تدريب قانونية
  • 6) تخصص: Web / Network / DFIR / Cloud / SOC
  • 7) بناء بورتفوليو: تقارير، لقطات، توثيق

قاعدة ذهبية

الاشتغال يكون قانوني وبموافقة صريحة فقط. أي تدريب أو اختبار يكون على بيئة تملكها أو مُصرّح لك بيها. الهدف: حماية الناس والأنظمة.

نصيحة: خليك تكتب “ملاحظات مختبر” لكل تجربة: شنو سويت؟ شنو تعلمت؟ شنو النتيجة؟

الأنظمة والبيئة (مختبر آمن)

أفضل طريقة تتعلم بيها بشكل صحيح: تبني مختبر منعزل. هذا يخليك تتدرب بدون ما تأثر على جهازك الأساسي.

Kali Linux

نظام متخصص للتدريب والتحليل الأمني، يحتوي أدوات فحص، مراقبة، وتحليل حركة الشبكة.

تحميل النظام الرسمي

VirtualBox

برنامج لتشغيل أنظمة وهمية (VM) حتى تسوي تدريب واختبارات داخل بيئة معزولة وآمنة.

تحميل البرنامج

الأدوات — شرح سهل + مثال تيرمنال

هاي الأدوات هي “عدة الشغل” بالمختبر. كل أداة إلها وظيفة محددة. هنا شرح بسيط + مثال بشكل تيرمنال (ضمن بيئة قانونية/مختبر).

Nmap

فكرته: “يفهم شنو موجود بالشبكة” — أجهزة فعّالة، منافذ مفتوحة، وخدمات شغّالة.

Network Recon
Terminal — Example (Lab)
$ nmap -sV 127.0.0.1 # يفحص الخدمات على جهازك (مختبر محلي) # -sV: معرفة إصدارات الخدمات

Burp Suite

فكرته: “وسيط” بين المتصفح والموقع، تراقب الطلبات (Requests) وتفهم منطق التحقق والصلاحيات.

Web Testing
Tip — Clean Workflow
# شغّل Burp ثم: # Browser -> Proxy -> Intercept ON # افحص: هل الطلب يحتوي Token؟ هل الصلاحيات تتحقق؟

Wireshark

فكرته: “يشوف الشبكة وهي تتحرك” — تحليل الترافيك، DNS، HTTP، TLS… مفيد للتشخيص والفهم.

Traffic Analysis
Filters (Wireshark)
# أمثلة فلاتر: dns http tcp.port == 443 ip.addr == 192.168.1.10

WhatWeb

فكرته: “يعرف تقنية الموقع” — سيرفر، CMS، مكتبات، بصمة عامة تساعدك تفهم البيئة.

Fingerprinting
Terminal — Example
$ whatweb https://example.com # بصمة عامة لموقع عام (للشرح)

Nikto

فكرته: فحص إعدادات الويب الشائعة والأخطاء المعروفة (Misconfigurations) — ممتاز للمراجعة السريعة داخل المختبر.

Web Review
Terminal — Local Lab
$ nikto -h http://127.0.0.1 # افحص سيرفر محلي عندك بالمختبر

Dirsearch

فكرته: يبحث عن مسارات (Paths) وملفات عامة/منسية داخل تطبيق ويب — مفيد لفهم هيكل الموقع داخل بيئة تدريب.

Content Discovery
Terminal — Safe Demo
$ dirsearch -u http://127.0.0.1 -e php,html,txt # استخدمه على سيرفرك المحلي بالمختبر

ملاحظة مهمة: الأمثلة هنا هدفها تعليمي داخل مختبرك أو بيئات مصرح بها. لا تستخدم الأدوات على أهداف حقيقية بدون إذن واضح.
